Setup Guide

How to get your site live on Netlify — no coding needed.

Step 1 — Upload files to GitHub
Go to github.com → your repo → upload all these files keeping the same folder structure. Make sure fics/index.json exists (upload the one included in the package).
Step 2 — Connect to Netlify
Go to netlify.com → Add new site → Import from GitHub → select your repo → Deploy. Done. Your site is live.
Step 3 — Using the admin panel
This admin panel saves fics to your browser. To publish them to your live site, click Export fics.json below, then upload that file to your GitHub repo at fics/index.json. Netlify auto-rebuilds within ~30 seconds.
That's it! Your workflow is: write fic here → Publish → Export JSON → upload to GitHub → site updates. You only touch GitHub for that one file upload step.
